openmodelica_microgrid_gym.util.itertools_.flatten(data, remaining_levels=0)[source]¶

transform this:

>>> {'a': {'b': [['i', 'v'],
>>>              ['k', 'h']]}}

results into

>>> ['a.b.i', 'a.b.v', 'a.b.k', 'a.b.h']

or

>>> [['a.b.i', 'a.b.v'], ['a.b.k', 'a.b.h']]
Parameters

  • data (Union[dict, list]) – data to flatten. A nested dictionary containing nested lists as keys in the lowest level.

  • remaining_levels (int) – number of levels to preserve in the nested list

list

openmodelica_microgrid_gym.util.itertools_.flatten_together(structure, values)[source]¶

Flattens and fills a list of values relative to the groupings provided by the structure parameter. The explicit values in the structure parameter are ignored. Only the nesting structure is of importance. If a single value is provided it is simply repeated as often as the structure has values

e.g. when called with [[0, 0], [0, 0]] and [[0, None], 4] it will detect the grouping and return [0, None, 4, 4]

Parameters

  • structure (List[Union[List, Any]]) – nested list used as a template

  • values (Union[Any, List[Union[List, Any]]]) – values matched to the list

Returns

flattened and filled list of values
